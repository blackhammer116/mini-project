Loops


  1. While loop

    • A while loop statement repeatedly executes a target statement as long as a given condition is true.
    • in order for a while loop to run effectively it needs to have three things, a condition, an expression and break case(in order to make the condition false and to avoid an infinite loop)

  2. Do-while loops

    • A do...while loop is similar to a while loop, except the fact that it is guaranteed to execute at least one time.
    • Unlike for and while loops, which test the loop condition at the top of the loop, the do...while loop in C programming checks its condition at the bottom of the loop.
    Syntax
    • Notice that the conditional expression appears at the end of the loop, so the statement(s) in the loop executes once before the condition is tested.
    • If the condition is true, the flow of control jumps back up to do, and the statement(s) in the loop executes again. This process repeats until the given condition becomes false.
    Output

  3. For loop

    • For this loop to work correctly it needs three things initialization, condition, and an update
    • initialization - initializes variables and is executed only once
    • condition - if true, the body of for loop is executed if false, the for loop is terminated
    • update - updates the value of initialized variables and again checks the condition
